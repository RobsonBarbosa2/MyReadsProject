MyReads Project
==============

This is a project made for NanoDegree from Udacity from React.

### Installation

1. Clone this project with `git clone https://github.com/RobsonBarbosa2/MyReadsProject.git`
2. Enter the project folder and install the dependencies with `npm install`
3. Run the application with `npm start`
*Attention: You must have installed on your computer the node and the npm*


### Libraries used

1. react-toastify
2. react-throttle
